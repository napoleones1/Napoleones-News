# Napoleones-News
Napoleones-News By: Muhamad Haikal
Napoleones News
Proyek Napoleones News dibuat sebagai bagian dari penyelesaian tugas mata kuliah Pemrograman Web. Proyek ini bertujuan untuk mengembangkan sebuah platform berita sederhana yang dilengkapi dengan fitur validasi form komentar menggunakan JavaScript, serta tampilan antarmuka yang responsif.

Tentang Proyek
Napoleones News adalah platform berita simulasi yang menghadirkan informasi terkini melalui antarmuka yang sederhana dan mudah dipahami. Proyek ini dirancang untuk memenuhi kebutuhan tugas akademik sekaligus mengasah keterampilan pemrograman web.

Tujuan
Mengimplementasikan form interaktif menggunakan HTML, CSS, dan JavaScript.
Meningkatkan pemahaman tentang validasi input di sisi klien.
Menyelesaikan tugas pemrograman web dengan menambahkan fitur logika sederhana seperti popup dan umpan balik pesan.

Fitur Utama
Untuk menambahkan fitur seperti di gambar yang kamu unggah, berikut beberapa fitur tambahan yang bisa dimasukkan ke dalam proyek Napoleones News:
Fitur Tambahan
Bagian "Terpopuler", "Latest", dan "Pilihan Kami"
Menampilkan berita dengan kategori berbeda secara dinamis.
Menggunakan gambar dan judul berita untuk memikat perhatian pengguna.
Kategori Berita (WORLD, SPORT)
Menampilkan ringkasan berita berdasarkan kategori tertentu.
Desain dengan warna yang berbeda untuk membedakan tiap kategori (misal, merah untuk WORLD, hijau untuk SPORT).
Recent News (Berita Terkini)
Bagian sidebar yang memuat daftar berita terbaru dengan judul singkat dan gambar thumbnail.
Judul berita terhubung ke halaman lengkapnya (menggunakan tag <a>).
Kolom Komentar Interaktif
Form komentar yang dilengkapi validasi (seperti yang sudah dibuat sebelumnya).
Menampilkan notifikasi popup untuk komentar sukses atau gagal.
Bagian Footer: About dan Instagram
About: Penjelasan singkat tentang platform berita.
Instagram: Galeri foto mini yang bisa dilihat lebih lanjut di akun Instagram.

Teknologi yang Digunakan
HTML5: Untuk struktur halaman.
CSS3: Untuk gaya dan tata letak.
JavaScript: Untuk validasi form dan pengaturan interaksi pengguna.

Cara Menggunakan
Clone repository ini ke komputer lokal kamu:
Copy code
git clone https://github.com/username/napoleones-news.git
Buka file index.html di browser.
Isi form komentar pada halaman dan klik tombol "Kirim Komentar".
Jika form kosong, pesan error akan muncul di halaman dan popup.
Jika form terisi, pesan sukses akan muncul, dan form akan otomatis dikosongkan.
Eksplorasi bagian lain dari halaman seperti tombol sosial media atau antarmuka lainnya.

Tujuan Akademik
Proyek ini dibuat sebagai simulasi pengembangan platform berita untuk mendemonstrasikan kemampuan:
Membuat validasi form dengan JavaScript.
Mendesain antarmuka responsif menggunakan HTML dan CSS.
Menerapkan logika sederhana untuk interaksi pengguna.
